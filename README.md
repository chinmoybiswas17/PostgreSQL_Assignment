1.What is PostgreSQL?
Ans:PostgreSQL হলো একটি ওপেন-সোর্স রিলেশনাল ডেটাবেস ম্যানেজমেন্ট সিস্টেম (RDBMS)।এটি SQL (Structured Query Language) ভিত্তিক এবং ডেটা সংরক্ষণ, অনুসন্ধান, আপডেট ও বিশ্লেষণে ব্যবহৃত হয়। এটি শক্তিশালী কুয়েরি ও জটিল ডেটা হ্যান্ডলিংয়ের ক্ষমতা রাখে। এটি সম্পূর্ণ ফ্রি ও ওপেন সোর্স। এছাড়াও ব্যবহারকারীরা তাদের নিজস্ব ডেটা টাইপ, অপারেটর এবং ফাংশন সংজ্ঞায়িত করতে পারে।
